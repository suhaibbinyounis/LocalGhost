# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-21

### Added
- Initial release
- Cross-platform service support (Windows, macOS, Linux)
- HTTP and WebSocket transports via FastAPI
- System tray integration with pystray
- Consent-driven authorization with native OS dialogs
- Token-based authentication with Fernet encryption
- SQLite permission storage with audit logging
- Auto-start on pip install
- Built-in demo plugin
- Interactive demo page at `/demo`
- CLI commands: run, install, uninstall, start, stop, status, permissions, revoke

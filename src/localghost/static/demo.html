Base URL

http://127.0.0.1:51473

LocalGhost always binds to localhost only for security.

Authentication

Protected endpoints require authorization. When accessing a protected endpoint for the first time:

  1. A native OS dialog appears asking for permission
  2. User approves: Allow Once, Session, or Always
  3. Token is stored and reused for future requests

Headers

Header Description Required
X-Process-Name Identifies your application Recommended
X-Process-PID Process ID for unique identification Optional
X-Client-ID Override automatic client ID Optional
Authorization Bearer token from previous approval For protected

Built-in Endpoints

Endpoint Method Type Description
/health GET Public Health check
/capabilities GET Public List all plugins and endpoints
/permissions GET Protected List granted permissions
/ws WebSocket Public Real-time communication

Demo Plugin Endpoints

Endpoint Method Type Description
/demo/ping GET Public Simple ping
/demo/echo POST Public Echo request body
/demo/time GET Public Current server time
/demo/system-info GET Protected System information
/demo/execute POST Protected Execute safe command

JavaScript Example

// Simple fetch request
const response = await fetch('http://127.0.0.1:51473/health');
const data = await response.json();

// With custom headers for identification
const response = await fetch('http://127.0.0.1:51473/demo/system-info', {
    headers: {
        'X-Process-Name': 'my-app',
        'X-Process-PID': '12345'
    }
});

// POST request
const response = await fetch('http://127.0.0.1:51473/demo/echo', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ message: 'Hello!' })
});

// WebSocket connection
const ws = new WebSocket('ws://127.0.0.1:51473/ws');
ws.onmessage = (e) => console.log(JSON.parse(e.data));
ws.send(JSON.stringify({ type: 'hello' }));

Python Example

import httpx

# Simple request
response = httpx.get("http://127.0.0.1:51473/health")
print(response.json())

# With identification headers
headers = {"X-Process-Name": "my-python-app"}
response = httpx.get("http://127.0.0.1:51473/demo/system-info", headers=headers)

# POST request
response = httpx.post(
    "http://127.0.0.1:51473/demo/echo",
    json={"message": "Hello from Python!"}
)

üõë How to Stop LocalGhost

Method Command
Press Ctrl+C in terminal Stops the running server
CLI command localghost stop
Kill by port lsof -ti :51473 | xargs kill (macOS/Linux)
System tray Click tray icon ‚Üí Quit
Kill all pkill -f localghost

CLI Commands

# Run server (with system tray)
localghost run

# Run without tray
localghost run --no-tray

# Stop the service
localghost stop

# Install as auto-start service
localghost install

# Manage service
localghost start
localghost stop
localghost status

# View permissions
localghost permissions

# Revoke access for a client
localghost revoke <client_id>

# Uninstall auto-start
localghost uninstall
